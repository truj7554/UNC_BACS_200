How to Make a Search Bar

Step 1: HTML Code Required

A. Using Ctrl+U look at the code for this website and copy the HTML for the following Searchbar

Home About Contact

B.Post that html into brackets and see how it comes out! Notice that it is very bare bones without any CSS in your browser.

Step 2: Edit your searchbar using CSS

Notice how my searchbar has some added flare. By copying my teach.css file you can find some easy ways to edit your searchbar. Try adding CSS to your page and playing with the colors! Some suggested edits by w3 schools are included in the code.
